## 1. install kernel headers

编译驱动需要依赖内核树，有两种方法：

* 安装内核头文件，编译时引用该头文件目录

```console
sudo apt-get install raspberrypi-kernel-headers
```

* 下载对应内核源码，编译时引用该内核文件目录。（下载内核源码版本一定要与目标平台一致）
