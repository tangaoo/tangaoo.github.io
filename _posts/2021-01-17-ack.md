**`Ack`** 工具用来做源代码中字符串查找（实则是对 `grep` 工具在程序源码查找方面的优化）。




{% raw %}

## Install

```
sudo aptitude install ack-grep
sudo ln -s `which ack-grep` /bin/ack
```

## Example
    
```
ack hello          #查找hello字符串
ack -a hello       #取消默认忽略
ack --js hello     #查找某特定语言
ack --nojs hello   #排除某特定语言
```

## Others

**`ack`** 默认忽略许多文件格式，下面命令查看具体哪些。

```
ack --help-types|less
```
